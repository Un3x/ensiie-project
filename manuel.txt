Manuel concernant les pages de src:

Alors, pour gérer les entrées de la table "User" par exemple il y a plusieurs fonctions:

chaque entrée est un objet vu par php donc pour récupérer les informations d'un objet il faut utiliser des 
fonctions du même style que $user->getNom(); qui récupère le nom de l'utilisateur user (selon les fonctions 
get que la table possède et qui sont dans le fichier User.php dans src par exemple)



@  pour récupérer la liste des éléments d'une table on récupère une liste d'objets :

$listetrucs = $trucRepository->fetchAll();
(si vous insérez ou modifiez la table il faut actualiser la liste en refaisant un $listetrucs = $trucRepository->fetchAll(); )



@  pour modifier un objet il faut utiliser les fonctions du type $truc->setMachin(parametre);
(les fonctions sont dans le truc.php dans src)


@  pour update un objet que vous avez modifié dans la table SQL il faut faire :

$trucRepository->updateTruc($truc);
/!\ il ne faut pas avoir modifié l'id qui doit rester unique


@  pour créer un nouvel objet "truc" il faut utiliser :

$lenomquevousvoulez = $trucRepository->creeTruc(parametres);
(avec les parametres en fonction de l'objet à voir dans src/trucRepository)


@  enfin pour insérer un nouvel objet que vous avez créé (avec la ligne juste au dessus) il faut utiliser :

$trucRepository->insertTruc($lenomquevousvoulez);
